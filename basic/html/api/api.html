Weather docs

CORS is enabled allowing Javascript make requests across domain boundaries
API URL ==> https://api.codetabs.com/api/weather/temp/
JSON format is used by default. To get data in XML set up format=xml .
Petitions are limited to 5 per second. You will get a 503 error if you exceed
Get your local temp : This endpoint retrieves actual temp based on your public IP geolocation.
HTTP Request : GET https://api.codetabs.com/api/weather/temp/
Query Parameters :
"&format=" values are json(by default or if parameter is empty) or xml 
  https://api.codetabs.com/api/weather/temp/
  https://api.codetabs.com/api/weather/temp?format=xml
Response : 
  {   "tempC": 21.62,
      "tempF": 53.62,
      "city": "Paris",
      "country": "FR"    }
Get temp from any location : This endpoint retrieves actual temp from the city passed as a parameter.
HTTP Request : GET https://api.codetabs.com/api/weather/temp?city=paris/
Query Parameters :
"&format=" values are json(by default or if parameter is empty) or xml
"&city=" city i want to know the temperature. 
  https://api.codetabs.com/api/weather/temp?city=paris
  https://api.codetabs.com/api/weather/temp?city=paris&format=xml
Response : 
  {   "tempC": 21.62,
      "tempF": 53.62,
      "city": "Paris",
      "country": "FR"    }
