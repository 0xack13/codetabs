View Headers
Tool to get list of response headers including redirect chain of a HTTP connection
Petitions are limited to 1 per second. You will get a 503 error if you exceed
Get list of response headers : This endpoint retrieves list of response headers including redirect chain of a HTTP connection in JSON format
HTTP Request :
GET https://api.codetabs.com/http-headers/get/urlToScan
Example
https://api.codetabs.com/http-headers/get/codetabs.com
