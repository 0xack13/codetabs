Welcome to codetabs.com/geoip API, a free GeoIP service that provides a public HTTPS (SSL access) API to retrieve geolocation from any IPv4, IPv6 or hostname

API doc

HTTP Request template:
GET https://api.codetabs.com/geoip/geoip/{format}?q={IP-or-hostname}
Supported formats : json, xml and csv
If no IP or hostname is provided it retrieves your own IP
IPv4 and IPv6 supported

Examples:
- https://api.codetabs.com/geoip/json
- https://api.codetabs.com/geoip/json?q=codetabs.com
- https://api.codetabs.com/geoip/xml?q=212.24.108.85
- https://api.codetabs.com/geoip/xml?q=2a00:1450:4006:803::200e

Usage Limits:
120 requests per minute. Once reached subsequent requests will result in error 503 (Service unavailable) until your quota is cleared
If you need more quota contact us
Our API requires no key or signup.
